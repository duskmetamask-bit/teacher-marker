# Place Value Unit: Understanding Our Number System
**Unit Duration:** 6 weeks × 3 lessons per week = 18 lessons
**Year Level:** Years 3–4
**Curriculum Link:** Australian Curriculum Mathematics v9 — Number
**Focus:** Place value to 10,000, partitioning, renaming, comparing and ordering
**Cold Task:** Pre-assessment — represent a number in multiple ways
**Hot Task:** Post-assessment — same task, measured for growth

## Unit Overview

**Learning Intention:**
We are learning to understand, represent and use place value to partition, compare and order numbers up to 10,000.

**Success Criteria:**
- I can read and write numbers to 10,000 in standard form and expanded form
- I can partition numbers in multiple ways (e.g. 3,482 = 3,000 + 400 + 80 + 2)
- I can rename numbers using different groupings (e.g. 340 = 34 tens or 340 ones)
- I can compare and order numbers to 10,000 using place value
- I can round numbers to the nearest 10, 100 or 1,000
- I can explain why place value works the way it does

## WEEK 1 — PLACE VALUE TO 1,000

### Lesson 1: Cold Task (Assessment)
**Time:** 40 minutes | **Type:** Assessment

| Phase | Teacher (I Do) | Students |
|-------|----------------|---------|
| Hook (5 min) | Show 347 on the board. Ask: "What does this number really mean? What does each digit represent?" | Think quietly |
| Task (35 min) | Cold Task: Show 562 in as many ways as you can. Think of every representation you know. | Independent representation task |

**Note the range of strategies shown.** Use this to differentiate teaching in Week 1.

---

### Lesson 2: Building Numbers (I Do)
**Time:** 50 minutes | **Type:** Explicit Teaching

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (15 min) | MAB (Base 10 blocks): Show 1 hundreds flat, 4 tens rods, 7 unit cubes. Ask: "What number is this? How do you know?" Build 347. Repeat with 293, 450, 508. | Observe, verify |
| Mini-Lesson (15 min) | Place value columns: HTU (Hundreds / Tens / Units). Each column is worth 10× the column to its right. Model on place value board. | Note the pattern |
| We Do (10 min) | Build these numbers with MAB together: 429, 306, 571, 880. Say the number aloud in words. | Build and say |
| You Do (10 min) | Build 3 numbers of your choice using MAB. Write the number in digits AND words. | Independent MAB building |

---

### Lesson 3: Partitioning (We Do)
**Time:** 50 minutes | **Type:** Guided Practice

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (10 min) | 347 = 300 + 40 + 7. But also = 340 + 7 = 30 tens + 7 = 347 ones. Show all four on the board. | Observe different partitions |
| We Do (20 min) | Partition challenge: "Show me 246 in at least 4 different ways." Use MAB, number expanders, and mental partitioning. Share strategies. | Find multiple partitions |
| You Do (20 min) | Choose 3 numbers. For each, show: standard partition, expanded form, and one creative partition. Prove all equal the same number. | Independent partitioning |

## WEEK 2 — PLACE VALUE TO 10,000

### Lesson 1: Extending to 4 Digits (I Do)
**Time:** 50 minutes | **Type:** Explicit Teaching

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (15 min) | Add a thousands column. MAB flat = 1,000. Build 2,500 together. What does each digit represent? Label: Th HTU. | Observe new scale |
| Mini-Lesson (10 min) | Pattern: 10 units = 1 ten. 10 tens = 1 hundred. 10 hundreds = 1 thousand. The pattern NEVER stops. It just keeps going. | Note the pattern |
| We Do (15 min) | Build and say: 1,234 / 3,007 / 4,560 / 5,000 / 6,789. Read each number aloud in words. | Build and read |
| You Do (10 min) | Write 5 four-digit numbers in words. Then swap with a partner and read theirs back. | Writing and reading practice |

---

### Lesson 2: Place Value Positions (I Do)
**Time:** 50 minutes | **Type:** Explicit Teaching

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (15 min) | Digit vs Digit VALUE: In 4,382 — the 3 is in the hundreds column, so its value is 300. In 3,482 — the 3 is in the thousands column, so its value is 3,000. Same digit, different value. | Grasp the distinction |
| Mini-Lesson (10 min) | Place value table: 4-digit grid (Th | H | T | U). Put 7,295 in the table. Ask: What does each digit contribute? |
| We Do (15 min) | Place Value Clues: "I'm thinking of a number. The digit 5 is in the hundreds place. The digit 8 is in the thousands place. The digit 3 is in the tens place. The digit 1 is in the units place. What number am I?" → 8,531. Make more together. | Solve place value riddles |
| You Do (10 min) | Make up 3 place value riddles for a partner. Solve partner's riddles. | Create and solve riddles |

---

### Lesson 3: What's Each Digit Worth? (You Do)
**Time:** 50 minutes | **Type:** Independent Practice

| Phase | Teacher | Students |
|-------|---------|---------|
| Mini-Lesson (5 min) | Face value (what the digit looks like) vs Place value (what it's worth) vs Total contribution (digit × column value). Example: In 6,134: digit 6 = face value 6, place value 6 × 1,000 = 6,000 total. | Note definitions |
| You Do (45 min) | Complete "Digit Worth" worksheet: For each number, identify face value, column, and total contribution for the underlined digit. | Independent work |

## WEEK 3 — COMPARING AND ORDERING

### Lesson 1: Greater Than, Less Than (I Do)
**Time:** 50 minutes | **Type:** Explicit Teaching

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (15 min) | Compare 3,847 vs 4,238. Rule: Start from the LEFT — highest place value first. Same thousands? No. 4,000 > 3,000. So 4,238 > 3,847. Model this thinking every time. | Learn the rule |
| Mini-Lesson (10 min) | Symbol practice: > (greater than) / < (less than) / = (equal to). Mnemonic: the big mouth (>) always faces the bigger number. | Remember symbols |
| We Do (15 min) | Compare pairs: 5,293 vs 5,923 / 7,104 vs 7,041 / 2,999 vs 3,000. Use place value boards to compare when close. | Compare systematically |
| You Do (10 min) | Write 3 comparison statements. Then write a word problem that results in one number being greater than another. | Apply to context |

---

### Lesson 2: Ordering Large Numbers (We Do)
**Time:** 50 minutes | **Type:** Guided Practice

| Phase | Teacher | Students |
|-------|---------|---------|
| We Do (20 min) | Race times task: 5 runners with times in milliseconds. Order them fastest to slowest. Requires careful comparison of 4-digit numbers. | Order by comparison |
| We Do (20 min) | Population task: 8 Australian towns with populations. Order largest to smallest. Discuss: Why do we need to read big numbers accurately? | Order real data |
| You Do (10 min) | Write your own ordering challenge for a partner with at least 5 four-digit numbers. Solve partner's challenge. | Create ordering puzzle |

---

### Lesson 3: Number Lines (I Do)
**Time:** 50 minutes | **Type:** Explicit Teaching

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (15 min) | Plot 4-digit numbers on a number line: 2,000 — 5,000 with 1,000 intervals. Where does 3,500 sit? 2,750? 4,123? | Observe number line placement |
| Mini-Lesson (10 min) | Number line strategy: Find the two nearest "round" numbers (landmarks). Place the number between them. Refine. | Learn landmark strategy |
| We Do (15 min) | Number line placement: Place 4,672 on a line from 4,000 to 5,000. What landmarks help? How precise can you be? | Practice placement |
| You Do (10 min) | Draw a number line from 1,000 to 2,000. Place these numbers: 1,250 / 1,500 / 1,750 / 1,990. Label each one. | Independent number lines |

## WEEK 4 — RENAMING AND REGROUPING

### Lesson 1: Trading Games (We Do)
**Time:** 50 minutes | **Type:** Kinaesthetic Learning

| Phase | Teacher | Students |
|-------|---------|---------|
| We Do (50 min) | MAB Trading Game: In groups of 4, each student has a role: Banker / Builder / Recorder / Checker. Banker has unlimited MAB. Builder rolls a 4-digit target number and builds it with MAB. Then: "Trade!" — trade all tens for hundreds, all hundreds for thousands. How many of each do you end up with? First team to correctly rename and record wins. | Active trading activity |

---

### Lesson 2: Renaming with MAB (We Do)
**Time:** 50 minutes | **Type:** Guided Practice

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (10 min) | Rename 4,300: How many ways? 4,300 = 43 hundreds = 430 tens = 4,300 ones = 4 thousands + 300. Model on the board. | See renaming patterns |
| We Do (20 min) | Rename Challenge: Use MAB to show ALL possible renamings of: 3,500 / 5,200 / 6,030 / 7,004. Record each renaming set. | Rename systematically |
| You Do (20 min) | Without MAB — rename these numbers in at least 3 different ways: 2,400 / 4,800 / 6,500. Explain your thinking to a partner. | Mental renaming |

---

### Lesson 3: Flexible Partitioning (I Do)
**Time:** 50 minutes | **Type:** Explicit Teaching

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (15 min) | 4,562 — Standard: 4,000 + 500 + 60 + 2. But also: 4,000 + 560 + 2. 3,000 + 1,562. 4,500 + 62. Show WHY these all work — they still equal 4,562. | See flexibility |
| Mini-Lesson (10 min) | When we add, we can partition numbers flexibly: 3,400 + 2,600 = (3,000 + 2,000) + (400 + 600) = 5,000 + 1,000 = 6,000. This is mental addition strategy. | Apply to addition |
| We Do (15 min) | Mental addition: 2,300 + 3,700 / 4,500 + 2,500 / 5,800 + 1,200. Use flexible partitioning — break it up YOUR way. | Practice mental addition |
| You Do (10 min) | Write a story problem that requires adding two 4-digit numbers. Solve it using flexible partitioning. | Apply to word problems |

## WEEK 5 — ROUNDING

### Lesson 1: Rounding to Nearest 10 (I Do)
**Time:** 50 minutes | **Type:** Explicit Teaching

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (15 min) | Rounding rule: Look at the digit ONE PLACE TO THE RIGHT of your target. 5 or more → round UP. 4 or less → round DOWN. Model: 47 → rounded to nearest 10: look at 7 (ones). 7 ≥ 5 → round up to 50. | Learn the rule |
| Mini-Lesson (10 min) | Number line for rounding: 47 sits between 40 and 50. It's CLOSER to 50. Draw it to see it. | Visualise on number line |
| We Do (15 min) | Round to nearest 10: 63 / 28 / 45 / 81 / 35 / 57 / 94 / 12. Check on a number line to verify. | Practice with number lines |
| You Do (10 min) | Real-world rounding: "I'm buying items that cost $38, $47 and $23. About how much will I spend?" → Round each to nearest 10. | Real-world application |

---

### Lesson 2: Rounding to Nearest 100 (We Do)
**Time:** 50 minutes | **Type:** Guided Practice

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (10 min) | Extend the rule: To round to the nearest 100, look at the TENS digit (two places right). Example: 364 → look at 6 in the tens place. 6 ≥ 5 → round up to 400. | Apply rule to hundreds |
| We Do (20 min) | Round to nearest 100: 250 / 349 / 450 / 873 / 1,249 / 2,380 / 4,550 / 7,891. Check on number line. | Practice |
| You Do (20 min) | Estimation challenge: "A school has 1,347 students. Each needs a $12 workbook. About how much will this cost?" → 1,300 × $12 ≈ estimate before calculating. | Estimate in context |

---

### Lesson 3: Rounding to Nearest 1,000 (You Do)
**Time:** 50 minutes | **Type:** Independent

| Phase | Teacher | Students |
|-------|---------|---------|
| I Do (10 min) | Rule: Look at the HUNDREDS digit (three places right). 5 or more → round up. Example: 3,750 → look at 7 in hundreds. 7 ≥ 5 → round up to 4,000. | Model rule |
| You Do (40 min) | Independent practice: Round to nearest 1,000: 2,499 / 3,500 / 4,372 / 5,899 / 6,501 / 7,250 / 8,999 / 9,801. Then: Real-world estimation problem set involving large numbers. | Independent and applied |

## ADAPTATIONS

### EAL Students
- Place value mat with visual supports (colour-coded columns)
- Number cards for each digit — can physically reorder to show digit value
- Sentence starters: "The digit __ is in the __ column. Its value is __."
- Pair with partner for MAB activities
- Pre-teach: "digit / column / worth / total" vocabulary

### Gifted Students
- Extend to 5-digit numbers (to 100,000) — explore millions column
- Inverse problem: Given the digit values, how many different numbers can you make? (e.g. 3,000 + 200 + 40 + 1 = 3,241 — what other arrangements of those digits make valid numbers?)
- Rounding to nearest 10 with decimals (e.g. round 4.7 to nearest whole number)
- Investigate: How do other number systems work? Binary place value introduction

### Students with NEPs
- Provide MAB manipulatives for all activities
- Reduce to 3-digit numbers — extend when ready
- Provide pre-drawn place value tables to fill in
- Allow calculator for checking after concept is understood
- Partner work for all games activities
- Reduce number of questions with emphasis on explanation quality over quantity
