Unit Overview

Learning Intention:

We are learning to use morphological knowledge (prefixes, suffixes, root words) to spell longer words, understand syllable division patterns, and distinguish homophones.

Success Criteria:

  • I can read and spell words with common prefixes (un-, re-, pre-, dis-, mis-, anti-, super-, auto-).
  • I can read and spell words with common suffixes (-tion, -sion, -ment, -ness, -ful, -less, -ly, -er, -est).
  • I can apply syllable division rules to read and spell longer words.
  • I can spell homophones correctly in context (their/there/they're, to/too/two, your/you're).
  • I can identify root words and apply spelling rules when adding suffixes.
  • I can use a dictionary to verify spellings.

    WEEK 1 — PREFIX INTRODUCTION

    Lesson 1: Cold Task (Assessment)

    Time: 45 minutes | Type: Assessment

    PhaseTeacher (I Do)Students
    Hook (5 min)"Matilda is a brilliant girl who discovers she has magical powers. Today we will test our spelling of words with prefixes, suffixes, and tricky syllable patterns."Engage
    Read (10 min)Read Chapter 1 of *Matilda*. Notice Dahl's spelling: "unhappy" (un-), "rewrite" (re-), "prehistoric" (pre-).Listen
    Task (30 min)Cold Task (same as Hot Task): 1. Write 6 words with prefixes: undo, rewrite, preview, dislike, misbehave, antiseptic. 2. Write 6 words with suffixes: happiness, breathless, quickly, teacher, tallest, wonderful. 3. Syllable division: pocket, rabbit, carpet, helicopter. 4. Homophones: their/there in a sentence. Year 3: All correct spelling expected.Independent spelling

    Differentiation: Prefix/suffix wall. Syllable division rules sheet. Homophone context chart.

    Lesson 2: un-, re-, pre- (I Do)

    Time: 45 minutes | Type: Explicit Teaching

    PhaseTeacher (I Do)Students
    Mini-Lesson (15 min)Prefix = letters added to the BEGINNING of a word to change its meaning. UN- = not (happy → unhappy). RE- = again (do → redo). PRE- = before (view → preview). The root word stays the same — the prefix just adds on. From *Matilda*: "unfair" (un-), "preview" (pre-), "retell" (re-).Learn
    Activity (20 min)Add prefixes to root words: happy → unhappy, kind → unkind, do → redo, write → rewrite, view → preview, heat → preheat, paid → repaid, move → remove. Write each. Circle the prefix. Say the meaning.Practice
    You Do (10 min)Write 5 sentences using prefixed words: "I will undo the knot." Circle each prefix.Written practice

    Success Criteria Check: Can add un-, re-, pre- to root words correctly.

    Lesson 3: dis-, mis-, anti- (We Do)

    Time: 45 minutes | Type: Guided Practice

    PhaseTeacherStudents
    We Do (30 min)Add prefixes: like → dislike, trust → distrust, behave → misbehave, understand → misunderstand, freeze → defrost (de- not dis), bacterial → antibacterial (anti-). Write each word. Explain the meaning. Sort into prefix groups.Guided
    You Do (15 min)Write 6 words with dis-, mis-, anti- prefixes. Write the root word and the prefixed word: dis + like = dislike. Circle the prefix.Written practice

    WEEK 2 — MORE PREFIXES

    Lesson 1: super-, auto-, multi- (I Do)

    Time: 45 minutes | Type: Explicit Teaching

    PhaseTeacher (I Do)Students
    Mini-Lesson (15 min)SUPER- = above, beyond (supernatural, superpower, superhero). AUTO- = self (autograph, autopilot, automatic). MULTI- = many (multiply, multipurpose, multimedia). These prefixes are less common but important for academic vocabulary.Learn
    Activity (20 min)Build words: super + star = superstar, super + hero = superhero, auto + graph = autograph, auto + mobile = automobile, multi + colored = multicolored, multi + cultural = multicultural. Write each. Identify the prefix.Building
    You Do (10 min)Write 5 sentences using super-, auto-, multi- words. Circle the prefix.Written practice

    Lesson 2: Prefix Spelling (I Do → We Do)

    Time: 45 minutes | Type: Explicit Teaching + Guided

    PhaseTeacherStudents
    I Do (10 min)When you add a prefix, the root word spelling NEVER changes. un + happy = unhappy (happy stays happy). re + write = rewrite (write stays write). pre + view = preview (view stays view). No letters are added or removed — just the prefix attached.Learn
    We Do (30 min)Predict then check: dis + trust, mis + understand, anti + bacterial, super + power, auto + mobile, multi + cultural. Write each word. Check that the root word is unchanged. Note any tricky spellings.Guided
    You Do (5 min)Write 8 prefixed words. Circle each prefix. Check: Did the root word change?Written practice

    Lesson 3: Prefixes in Context (You Do)

    Time: 45 minutes | Type: Independent

    PhaseTeacherStudents
    You Do (45 min)Dictation: "The unhappy student decided to rewrite her story. The superhero previewed the movie. I cannot distrust my best friend. The multicolored umbrella was automatic." 1. Write all prefixed words from the passage. 2. Write 4 sentences of your own using prefixed words. 3. Alphabetical order: redo, preview, misplace, superhuman, untie, preschool, anticlockwise, autopilot.Full practice

    WEEK 3 — SUFFIX INTRODUCTION

    Lesson 1: -ment, -ness, -ful (I Do)

    Time: 45 minutes | Type: Explicit Teaching

    PhaseTeacher (I Do)Students
    Mini-Lesson (15 min)Suffix = letters added to the END of a word to change its meaning or make a new word. -MENT = state of (achieve → achievement). -NESS = state of (happy → happiness). -FUL = full of (care → careful). Unlike prefixes, suffixes CAN change the root word's spelling. From *Matilda*: "adventure" (not a suffix word but shows root), "wonderful" (-ful).Learn
    Activity (20 min)Add suffixes: achieve → achievement, achieve → achieving, achieve → achieved. state → statement → statements. happy → happiness, happy → happily. care → careful → carefully. Note changes to root words.Practice
    You Do (10 min)Write 6 suffix words. Circle the suffix. Write the root word: enjoyment (enjoy), fulfillment (full), darkness (dark), treatment (treat).Written practice

    Lesson 2: -less, -ly, -er (I Do → We Do)

    Time: 45 minutes | Type: Explicit Teaching + Guided

    PhaseTeacherStudents
    I Do (10 min)-LESS = without (care → careless, hope → hopeless). -LY = in a certain way (quick → quickly, happy → happily). -ER = more, or one who (tall → taller, teach → teacher).Learn
    We Do (30 min)Add suffixes: hope → hopeless, hope → hopeful, hope → hopefully. quick → quickly, quick → quicker, quick → quickest. loud → loudly, loud → louder, loud → loudest. Write each word. Circle the suffix.Guided
    You Do (5 min)Write 4 -less, 4 -ly, 4 -er words. Circle the suffix.Written practice

    Lesson 3: Suffix Spelling (You Do)

    Time: 45 minutes | Type: Independent

    PhaseTeacherStudents
    You Do (45 min)1. Dictation: "The careless student was hopeless. The teacher spoke quickly to the helpful class." 2. Sort words: enjoyment, wonderful, colourful, reckless, happily, teacher, darkness, fearless, quietly, treatment. Sort into -ment, -ness, -ful, -less, -ly, -er columns. 3. Write 3 sentences using suffix words.Full practice

    WEEK 4 — SUFFIXES AND ROOT WORDS

    Lesson 1: Adding -ing, -ed (I Do)

    Time: 45 minutes | Type: Explicit Teaching

    PhaseTeacher (I Do)Students
    Mini-Lesson (15 min)Adding -ing and -ed to root words: MOST words just add -ing/-ed: jump → jumped, walk → walking. END with E: drop the E first, then add -ing/-ed: love → loving, love → loved. END with consonant-vowel-consonant (CVC): double the final consonant, then add -ing/-ed: tap → tapped, tap → tapping. From *Matilda*: "playing" (drop e), "tapping" (double p).Learn
    Activity (20 min)Apply the rules: love → loving, dance → dancing, hope → hoping, hop → hopped, plan → planned, tap → tapping, sit → sitting, run → running. Say which rule applies. Write each word.Practice
    You Do (10 min)Write 8 words with -ing and -ed. Circle any that required a spelling change (drop e or double consonant).Written practice

    Lesson 2: Doubling Rule (I Do → We Do)

    Time: 45 minutes | Type: Explicit Teaching + Guided

    PhaseTeacherStudents
    I Do (10 min)Doubling rule: If a word ends CVC and the last sound is STRESSED, double the consonant before a suffix starting with a vowel. hop → hopping (stressed). open → opening (not stressed — no double). From *Matilda*: "tapping" (tap → tapped, double p).Learn
    We Do (30 min)Apply: hop → hopped, hopping. skip → skipped, skipping. admit → admitted, admitting. open → opening, opened. visit → visited, visiting. prefer → preferred, preferring. Note which doubled and which didn't — explain why.Guided
    You Do (5 min)Write 6 words with doubling. Write the rule explanation: why did we double?Written practice

    Lesson 3: Dropping Silent e (You Do)

    Time: 45 minutes | Type: Independent

    PhaseTeacherStudents
    You Do (45 min)1. Complete: love → lo__ing, lo__ed; hope → hop__ing, hop__ed; dance → danc__ing, danc__ed; write → writ__ing, writ__ten; smile → smil__ing, smil__ed. 2. Dictation: "Matilda was hoping to love school. The teacher was smiling." 3. Sort: dropped-e words vs doubled-consonant words. 4. Write a sentence using one dropped-e word and one doubled-consonant word.Full practice

    WEEK 5 — SYLLABLE DIVISION

    Lesson 1: Closed Syllable Division (I Do)

    Time: 45 minutes | Type: Explicit Teaching

    PhaseTeacher (I Do)Students
    Mini-Lesson (15 min)Syllable division rules: VCCV (vowel-consonant-consonant-vowel): Divide BETWEEN the consonants. PIC-nic, BAS-ket, HAP-pen, RAB-bit. VCV (vowel-consonant-vowel): Usually divide AFTER the consonant — the first syllable stays CLOSED and short. RE-bel, PA-per, TA-ble, LO-cate.Learn
    Activity (20 min)Divide these words: pocket (pock-et), rabbit (rab-bit), carpet (car-pet), helmet (hel-met), basket (bas-ket), rocket (roc-ket), magnet (mag-net), comet (com-et). Say each syllable. Check: VCCV → divide between consonants.Practice
    You Do (10 min)Divide these words: chocolate, vegetable, opposite, magazine, separate, desperate, definite, fortunate. Write the syllable breaks.Written practice

    Lesson 2: Open Syllable Division (I Do → We Do)

    Time: 45 minutes | Type: Explicit Teaching + Guided

    PhaseTeacherStudents
    I Do (10 min)Open syllables (ending in vowel) are usually long: TA-ble (ta = /tay/), RO-bot (ro = /roe/), FU-ture (fu = /fyoo/). When dividing, find if the first part ends with a vowel (open) or consonant (closed). Open = long vowel, closed = short vowel.Learn
    We Do (30 min)Divide and identify: ta-ble (open, long), ro-bot (open, long), ba-by (ba is open, by is closed), pa-per (open, long), de-ny (de is closed /den/, ny = /nie/), re-mote (re is closed, mote is open). Note: The first syllable gives the first clue about vowel length.Guided
    You Do (5 min)Divide: music, unit, data, remote, deny, lady, open, final. Label each first syllable as open or closed.Written practice

    Lesson 3: VCV and VCCV Rules (You Do)

    Time: 45 minutes | Type: Independent

    PhaseTeacherStudents
    You Do (45 min)1. Divide these words correctly: chocolate, separate, decorate, certificate, hesitate, indicate, opposite, incredible, decorate, eliminate. Write syllable breaks. 2. Sort into VCV vs VCCV division: liberate, history, picture, capture, temple, memory, capture, another. 3. Dictation with divided words: "The chocolate was separate from the cake." "The student was a talented musician."Full practice

    WEEK 6 — HOMOPHONES

    Lesson 1: Their/there/they're (I Do)

    Time: 45 minutes | Type: Explicit Teaching

    PhaseTeacher (I Do)Students
    Mini-Lesson (15 min)Homophones = words that sound the same but have different spellings and meanings. THEIR = belonging to them (their books). THERE = a place (over there). THEY'RE = they are (they're coming). The word's SPELLING shows its MEANING — context is everything. From *Matilda*: "Their house was large." "The books were over there." "They're going to school."Learn
    Activity (20 min)Choose the correct word: "____ books are on ____ shelf." (Their, there) "____ going to eat ____ dinner at ____ house." (They're, their, there) "____ are ____ seats over ____." (There, they're, there). Write sentences with context.Practice
    You Do (10 min)Write 3 sentences using their/there/they're correctly. Underline the homophone in each.Written practice

    Lesson 2: To/too/two, your/you're (I Do → We Do)

    Time: 45 minutes | Type: Explicit Teaching + Guided

    PhaseTeacherStudents
    I Do (10 min)TO = direction or for (to school). TOO = also or excessive (me too, too hot). TWO = the number 2. YOUR = belonging to you (your book). YOU'RE = you are (you're welcome). "Your book" — it's YOUR book. "You're welcome" — YOU ARE welcome.Learn
    We Do (30 min)Complete: "I want to go ___ school and ___ Jane." (to, too) "___ going to ___ house to get ___ shoes." (You're, your, your) "Can ___ have a cookie ___? I have ___ cookies." (you, too, two) Write each sentence with correct spelling.Guided
    You Do (5 min)Write 3 sentences using to/too/two and 3 using your/you're. Underline the homophone.Written practice

    Lesson 3: Homophone Practice (You Do)

    Time: 45 minutes | Type: Independent

    PhaseTeacherStudents
    You Do (45 min)1. Complete all homophones: "The children walked ___ school. It was ___ hot ___ walk. ___ teacher was there. She said ___ doing well. ___ are good students." (to, too, their, There, You're, They're). 2. Write a paragraph (8+ sentences) using at least 6 different homophones correctly. 3. Create your own homophone pair quiz for a partner.Full practice

    WEEK 7 — ADVANCED AFFIXES

    Lesson 1: -tion, -sion, -ture (I Do)

    Time: 45 minutes | Type: Explicit Teaching

    PhaseTeacher (I Do)Students
    Mini-Lesson (15 min)Advanced suffixes: -TION says /shun/ (action, section, attention, education, celebration, introduction). -SION says /shun/ or /zhun/ (division, occasion, television, explosion). -TURE says /cher/ or /cher/ (adventure, picture, nature, culture, sculpture). These suffixes create noun forms from verbs and adjectives.Learn
    Activity (20 min)Complete: act → ac__, sec → sec__, edu__ → edu__, cele__ → cele__. divi__ → divi__, occa__ → occa__, adven__ → adven__, pic__ → pic__. Write each word. Say the suffix.Practice
    You Do (10 min)Write 6 -tion words and 4 -ture words. Circle the suffix.Written practice

    Lesson 2: -ous, -ious, -eous (I Do → We Do)

    Time: 45 minutes | Type: Explicit Teaching + Guided

    PhaseTeacherStudents
    I Do (10 min)-OUS, -IOUS, -EOUS say /us/ and mean "having the quality of." DANGEROUS (danger + ous), CURIOUS (curio + us), GLOBE (glo + beous → gorgeous). These create adjectives from nouns.Learn
    We Do (30 min)Complete: danger → danger__, curio → curi___, glo → __eous, fame → fam__, nerve → ner__, humor → hum__. Write each word. Check: -ous, -ious, or -eous?Guided
    You Do (5 min)Write 4 -ous, 4 -ious, 2 -eous words. Circle the suffix.Written practice

    Lesson 3: Advanced Spelling (You Do)

    Time: 45 minutes | Type: Independent

    PhaseTeacherStudents
    You Do (45 min)1. Dictation: "The celebration was a wonderful occasion. The beautiful sculpture was an adventure in nature. The dangerous situation required careful attention." 2. Sort: action, adventurous, precious, section, musician, culture, famous, education, dangerous, television. Sort into -tion, -ture, -ous, -ious columns. 3. Write a sentence using one word from each suffix group.Full practice

    ADAPTATIONS

    EAL Students

  • Provide prefix/suffix charts with meanings in L1
  • Focus on un-, re-, pre- before introducing others
  • Use word families to teach suffixes
  • Allow verbal spelling for complex morphological words
  • Provide homophone context sentences with L1 equivalents

    • Gifted Students

  • Introduce Greek/Latin roots (tele, graph, port, dict)
  • Explore more complex suffixes (-ologist, -ification)
  • Research etymology for spelling patterns
  • Create morphological word games
  • Explore advanced homophones (aloud/allowed, affect/effect)
  • Introduce spelling generalisations (-able, -ible)

    • Students with NEPs

  • Focus on un-, re-, pre- prefixes only
  • Focus on -ing, -ed, -er, -est suffixes only
  • Provide syllable division templates
  • Allow word bank use for all activities
  • Simplify homophones to their/there and to/too/two only
  • Extended time for dictation
